## What I'm building

### 🩺 [Sistola](https://github.com/christiangf28/sistola) — Blood Pressure Monitor for Android
Personal health app built with React Native + Expo. Tracks BP, sleep, stress, activity and medication adherence. Contextual AI insights via Claude API. Available on Google Play (testing track).

`React Native` `Expo` `TypeScript` `AsyncStorage` `Claude API` `EAS Build`

### 🏥 Centinela — Predictive Clinical Management Platform
Web platform for hospital directors and managers. Predicts length of stay (LOS), ICU admission risk, and generates automated executive reports from clinical datasets using AI.

`Python` `Data Analytics` `Predictive Modeling` `AI APIs` `HealthTech`
